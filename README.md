# 🔐 Python Network Port Scanner

A lightweight TCP port scanner built using Python and socket programming.  
This tool detects open ports on a target system and helps understand network exposure — an essential concept in cybersecurity.

---

## 🚀 Features

✔ Scan open TCP ports  
✔ Supports IP address & domain scanning  
✔ Fast and simple execution  
✔ Beginner-friendly cybersecurity tool  
✔ Tested in Kali Linux  

---

## 🛠 Technologies Used

- Python 3
- Socket Programming
- Kali Linux
- Networking Basics
  

## ▶️ How to Run :

Step 1: Open Terminal

Step 2: Navigate to project folder  with 
        cd python-port-scanner
        
Step 3: Run the program
        Python3 port_scanner.py

Step 4: Enter target IP or domain

## Example Output

port 80 is open  

Scan complete!
